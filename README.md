## Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```
   git clone https://github.com/Manoj5621/Stranger-Thiongs.git
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the development server:**
   ```
   npm run dev
   ```

Open your browser and navigate to the provided local URL to view the project.
